## 9. Optimization Rules

> Automated and manual rules for campaign management.

### Pause Rules (Stop Spending)

| Condition                                    | Action               | Review Period |
|----------------------------------------------|----------------------|---------------|
| CPA > R$ {{CPA_PAUSE_THRESHOLD}}             | Pause ad set         | {{PERIOD}}    |
| CTR < {{CTR_PAUSE_THRESHOLD}}%               | Pause creative       | {{PERIOD}}    |
| Frequency > {{FREQ_PAUSE_THRESHOLD}}         | Pause ad set         | {{PERIOD}}    |
| Spend > {{SPEND_THRESHOLD}} with 0 conversions | Pause ad set       | {{PERIOD}}    |
| ROAS < {{ROAS_PAUSE_THRESHOLD}}x             | Pause campaign       | {{PERIOD}}    |

### Scale Rules (Increase Budget)

| Condition                                    | Action               | Max Increase  |
|----------------------------------------------|----------------------|---------------|
| CPA < R$ {{CPA_SCALE_THRESHOLD}} for {{DAYS}} days | Increase budget {{SCALE_%}}% | {{MAX_DAILY}} |
| ROAS > {{ROAS_SCALE_THRESHOLD}}x for {{DAYS}} days | Increase budget {{SCALE_%}}% | {{MAX_DAILY}} |
| CTR > {{CTR_SCALE_THRESHOLD}}% consistently  | Duplicate and scale  | {{MAX_DAILY}} |

### Adjustment Rules (Modify Strategy)

| Condition                                    | Action                                      |
|----------------------------------------------|---------------------------------------------|
| Learning phase not exiting after {{DAYS}} days | Review audience size, increase budget by 20% |
| Ad fatigue (frequency > {{FREQ_FATIGUE}})    | Rotate creatives, refresh copy               |
| CPM rising > {{CPM_RISE_%}}% week-over-week  | Adjust targeting, test new placements        |
| Conversion rate dropping                     | Review landing page, check pixel firing      |

### Escalation
- If KPIs are consistently outside acceptable range for 3+ days, escalate to @marketing-chief.
- If budget burn rate exceeds 120% of planned pace, pause and notify immediately.

## 10. Reporting Schedule

### Report Cadence

| Frequency    | Report Type              | Recipient(s)                          | Delivery Method        |
|--------------|--------------------------|---------------------------------------|------------------------|
| **Daily**    | Performance snapshot     | @traffic-manager                      | Dashboard / automated  |
| **Weekly**   | Weekly performance report| @traffic-manager, @content-strategist | Document + meeting     |
| **Bi-weekly**| Mid-campaign review      | @marketing-chief, @traffic-manager    | Presentation           |
| **Monthly**  | Full campaign report     | @marketing-chief, all stakeholders    | Document + meeting     |
| **Post-campaign** | Final analysis      | @marketing-chief, executive team      | Formal report          |

### Report Contents

#### Daily Snapshot
- Spend vs. budget pace
- Key KPIs (CPA, ROAS, CTR)
- Top/bottom performing ad sets
- Anomalies or alerts

#### Weekly Report
- Week-over-week KPI comparison
- Budget utilization
- A/B test status and interim results
- Creative performance ranking
- Audience insights
- Optimization actions taken
- Plan for next week

#### Final Campaign Report
- Executive summary
- KPI results vs. targets (with variance analysis)
- Budget summary (planned vs. actual)
- A/B test results and learnings
- Top performing audiences, creatives, placements
- Recommendations for future campaigns
- Full data appendix
