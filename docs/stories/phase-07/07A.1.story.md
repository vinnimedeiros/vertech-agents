# Story 07A.1 — Confirmação de dependências externas

**Status:** Draft

**Owner agent:** `@analyst` (Link)

**Estimated effort:** 30-45 min

## Context

O design da Phase 07 (`docs/superpowers/specs/2026-04-19-phase-07-mastra-design.md`) referencia APIs externas cujos identificadores podem ter mudado entre a data do vault (2025) e a execução atual (2026-04). Antes que o @dev toque em `package.json` ou código, é crítico **confirmar versões e IDs exatos** pra evitar retrabalho e erros de API deprecated.

## Acceptance Criteria

1. **Model ID do OpenAI `gpt-4.1-mini`** confirmado via context7 — nome exato aceito por `@ai-sdk/openai`
2. **Catálogo de modelos OpenAI** disponíveis no AI SDK listado (GPT-4.1, GPT-4o, GPT-4o-mini, etc) com IDs exatos
3. **Catálogo de modelos Anthropic** disponíveis no AI SDK listado (Claude Haiku 4.5, Sonnet 4.6, Opus 4.7) com IDs exatos
4. **Package oficial Mastra** para storage Postgres confirmado — nome exato, versão atual, e API de `new PostgresStore()` ou equivalente
5. **Memory API do Mastra** (`@mastra/memory`) — confirmar `new Memory({ options: { lastMessages, observationalMemory } })` continua válido ou se houve breaking change
6. **BullMQ** — confirmar versão atual e pattern oficial de **serialização por conversa** (opções: `BullMQ Pro` com groupKey; ou pattern `jobId` único por `conversationId` + `maxJobsPerConversation: 1`; ou lock curto em Redis por `conversationId`)
7. **Bull-Board** — confirmar versão compatível com Next.js 15 App Router e pattern de montagem em rota `/admin/queues`
8. **React Flow** — confirmar versão atual + pattern de layout automático recomendado (dagre vs elkjs vs manual)
9. Documento entregue em `docs/phase-07/dependencies-confirmed.md` com tabela: `{package, version, install_command, confirmed_date, source_url, notes}`
10. Qualquer **divergência** do que está escrito no design spec (seção 12) deve ser **explicitamente destacada** no documento com proposta de ajuste
