# Story 07B.6 — Aba Conversas

**Status:** Done (aguardando gate humano no fim do bloco)
**Phase:** 07B (UI essencial)
**Owner agent:** `@dev` (Neo)
**Story type:** UI (form com textareas + 2 TagLists)
**Estimated effort:** 45-60 min

## Context

Quarta aba. Define como o agente **conversa**: saudação, qualificação, objeções, handoff. Inputs que modelam o fluxo de conversa no `buildInstructions`.

Reutiliza `TagList` da 07B.4 pra dois campos (perguntas de qualificação + triggers de handoff).

**Spec de referência:** `docs/phase-07/ui-spec-07b-agent-detail.md` — seção 7.4 (aba Conversas).

## Acceptance Criteria

### Page `/agents/[agentId]/conversation`

1. Criar `apps/web/app/(saas)/app/(organizations)/[organizationSlug]/agents/[agentId]/conversation/page.tsx` renderizando `<ConversationTab />`.
2. `ConversationTab` usa `useAgent()` + `useAgentForm({ schema: conversationSchema, defaultValues: agent.conversationStyle, action: updateAgentConversationStyle })`.
3. Título: "Conversas" + subtítulo: "Como o agente aborda a conversa — saudação, qualificação, objeções, quando chama humano."

### Campos

4. **Saudação padrão** (`conversationStyle.greeting`) — `Textarea` 2 rows, max 300 chars. Helper: "Primeira mensagem do agente. Ex: 'Oi! Sou a Maria da Vertech, posso te ajudar?'".
5. **Perguntas de qualificação** (`conversationStyle.qualificationQuestions`) — `TagList` (reutiliza da 07B.4). Array max 10 items, 200 chars cada. Helper: "Perguntas que o agente faz pra entender o cliente antes de oferecer algo. Ex: 'Qual o tamanho da empresa?', 'Qual dor principal vocês querem resolver?'".
6. **Tratamento de objeções** (`conversationStyle.objectionHandling`) — `Textarea` 4 rows, max 1000 chars. Helper: "Como o agente responde a 'tá caro', 'vou pensar', 'já tenho outro fornecedor'. Ex: 'Acolher com empatia, perguntar o porquê, trazer diferencial sem pressionar'".
7. **Gatilhos de handoff humano** (`conversationStyle.handoffTriggers`) — `TagList`. Array max 10 items, 150 chars cada. Helper: "Situações que transferem pra humano. Ex: 'cliente pedir humano', 'pergunta sobre valor específico fora das políticas', 'situação emocional delicada'".

### Comportamento

8. `DirtyStateBanner` no rodapé.
9. Submit:
    - `updateAgentConversationStyle({ agentId, conversationStyle })`.
    - Toast: "Estilo de conversa atualizado."
    - `revalidatePath` do detalhe.
10. TagList deve adicionar item via Enter OU botão "Adicionar". Botão remover (X) em cada chip.
11. Estado arquivado bloqueia edição.

### Schema zod

12. Adicionar em `lib/schemas.ts`:
    ```typescript
    export const conversationStyleSchema = z.object({
      greeting: z.string().max(300).optional().nullable(),
      qualificationQuestions: z.array(z.string().min(1).max(200)).max(10),
      objectionHandling: z.string().max(1000).optional().nullable(),
      handoffTriggers: z.array(z.string().min(1).max(150)).max(10),
    });
    export const conversationSchema = z.object({
      conversationStyle: conversationStyleSchema,
    });
    ```
