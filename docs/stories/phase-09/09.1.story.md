# Story 09.1 — Tela de boas-vindas `/agents` (Hero + grid 7 templates + SessionHistory)

**Status:** Draft
**Phase:** 09
**Owner agent:** `@dev` (Neo)
**Story type:** UI + server components
**Estimated effort:** 3-4h

## Context

Substitui a tela atual `/agents` (que era "Lista simples" na v1) pela nova tela de boas-vindas do Arquiteto. Dois estados: (a) vazio = hero dominante + grid de templates; (b) com agentes = hero comprimido + accordion "Rascunhos" + "Agentes criados".

Esta é a porta de entrada PRIMÁRIA da experiência de criação (PRD v2 + UI Spec § 3).

**Specs de referência:**
- UI Spec § 3 (wireframes desktop + mobile): `docs/phase-09/ui-spec-arquiteto.md`
- Tech Spec § 9.2 (packages afetados): `docs/phase-09/tech-spec-arquiteto.md`

## Acceptance Criteria

### Layout base e estados

1. GIVEN rota `/app/[orgSlug]/agents`, WHEN user sem agentes e sem rascunhos, THEN mostra layout "estado vazio": hero central max-w 640px + CTA "Criar com o Arquiteto" + divisor + grid 4+3 templates.
2. GIVEN user com agentes/rascunhos, WHEN renderiza, THEN layout "estado com agentes": PageHeader comprimido + Accordion "Rascunhos em andamento (N)" expandido + Accordion "Agentes publicados (M)" expandido + Accordion colapsado "Começar do zero com um template".
3. GIVEN breakpoint `lg:` (>=1024px), WHEN desktop, THEN grid de templates 4 cols. GIVEN `md:` (768-1023), THEN 3 cols. GIVEN mobile (<768px), THEN 2 cols + Personalizado na última linha sozinho.

### Hero

4. GIVEN estado vazio, WHEN renderiza hero, THEN título "Crie seu primeiro agente comercial em 15 minutos" (text-4xl font-semibold 2 linhas), subtítulo muted max-w 480px, CTA `Button size="lg"` com `Sparkles` icon + "Criar com o Arquiteto".
5. GIVEN estado com agentes, WHEN hero comprimido, THEN PageHeader padrão com título "Agentes" + subtítulo + CTA à direita "+ Criar novo agente".

### Grid de 7 templates

6. GIVEN grid de templates, WHEN renderiza, THEN 7 cards: Clínica 🏥, E-commerce 🛒, Imobiliária 🏠, Infoprodutor 💼, SaaS ⚙️, Serviços locais 🍽️, Personalizado ✨ (último com `border-dashed`).
7. GIVEN card template, WHEN renderizado, THEN estrutura: ícone 48px centralizado + label (text-base font-medium) + micro-copy 2 linhas text-xs muted.
8. GIVEN hover no card, WHEN mouse over, THEN `border-primary/50` + `scale-[1.02]` com transição 150ms.
9. GIVEN click no card, WHEN user seleciona, THEN navega pra `/app/[orgSlug]/agents/new?template={templateId}`.
10. GIVEN card Personalizado clicado, WHEN navega, THEN usa `templateId=custom`.

### Seção Rascunhos em andamento (Accordion)

11. GIVEN user com sessões DRAFT, WHEN accordion expandido, THEN lista cards horizontais com: avatar, nome do agente/template, "Etapa {stage} • há {tempo}", CTA "Continuar".
12. GIVEN card de rascunho clicado (nome ou CTA), WHEN navega, THEN vai pra `/agents/new?session={sessionId}`.
13. GIVEN sessão com `draftSnapshot.lastActivity` há 3h, WHEN renderiza, THEN mostra "há 3 horas". (Usar helper de formatação de tempo relativo existente ou criar.)
14. GIVEN 0 rascunhos, WHEN accordion, THEN mostra mensagem "Nenhum rascunho em andamento".

### Seção Agentes publicados (Accordion)

15. GIVEN user com agentes ACTIVE/DRAFT/PAUSED (não ARCHIVED), WHEN accordion expandido, THEN grid de cards reusando `AgentCard` da 07B.1 v1 (compatibilidade preservada).
16. GIVEN card agente clicado, WHEN navega, THEN vai pra `/agents/[agentId]` (Painel de Refino, que será implementado em 07B-v2 mas pode ter placeholder agora — fallback OK).

### Seção Templates (accordion colapsado)

17. GIVEN user com agentes, WHEN accordion "Começar do zero com um template" colapsado, THEN usuário pode expandir pra ver o grid de 7 templates (mesmo componente da estado vazio).

### Server logic

18. GIVEN rota é server component, WHEN renderiza, THEN carrega via `getAgentsByOrg(orgId)` (já existe 07B.1 v1), `getDraftSessions(userId, orgId)` NEW.
19. GIVEN `getDraftSessions`, WHEN query, THEN filtra `agent_creation_session` WHERE `userId = $1 AND organizationId = $2 AND status = 'DRAFT'`, ordena `updatedAt DESC`, limita 6 (UI mostra ver-todos se passar).

### Translations

20. GIVEN textos novos, WHEN i18n, THEN adicionar namespace `architect.welcome.*` em `pt-BR.json` (ver UI Spec § 14 chaves exatas).
